# Overview

Task-focused guides for the most common Bit2Me workflows.

These guides cover the currently implemented HTTP surface. WebSocket flows are not available yet.

## Included Guides

- [Fetch Market Data](fetch-market-data.md) - tickers, order books, market config, candles, and last trades
- [Place & Manage Orders](place-and-manage-orders.md) - create, list, inspect, cancel, and inspect order trades
- [Fetch Balances & Transactions](fetch-balances-and-transactions.md) - trading balances, wallet transactions, and earn wallets
- [Move Funds & Pockets](move-funds-and-pockets.md) - manage pockets and move funds between Bit2Me Wallet and Trading
- [Fetch Earn & Asset Data](fetch-earn-and-asset-data.md) - earn APY, earn assets, earn wallets, and currency assets
